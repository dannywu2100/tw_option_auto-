# AutoPlus1 Share Page

這個資料夾是給 GitHub Pages 用的公開分享版本。

## 內容

- `index.html`: 可直接發佈的完整教學頁
- `.nojekyll`: 避免 GitHub Pages 套用 Jekyll 處理

## 最簡單發佈方式

1. 在 GitHub 建一個新的公開 repo。
2. 把這個資料夾內的檔案上傳到 repo 根目錄。
3. 到 GitHub repo 的 `Settings -> Pages`。
4. Source 選 `Deploy from a branch`。
5. Branch 選 `main`，Folder 選 `/ (root)`。
6. 存檔後等待 GitHub Pages 發佈完成。

之後網址通常會是：

```text
https://你的帳號.github.io/你的-repo-名稱/
```
